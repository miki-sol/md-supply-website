# MD Supply — корпоративный сайт

Сайт-визитка торговой компании ООО «МД Сапплай» (MD Supply), г. Минск. Информационный сайт без интернет-магазина: представление компании, ассортимента и условий сотрудничества + приём заявок.

## Технологии

- **Next.js 16** (App Router, React 19.2) — SSR/SSG, SEO из коробки
- **TypeScript**
- **Tailwind CSS v4** — фирменная тема через CSS-токены
- **Framer Motion** — анимации появления, мега-меню, модалки
- **next-themes** — переключатель тёмной/светлой темы
- **react-hook-form + zod** — формы и валидация
- **lucide-react** — иконки

## Команды

```bash
pnpm install
pnpm dev      # http://localhost:3000
pnpm build    # production-сборка
pnpm start    # запуск собранного приложения
pnpm lint
```

## Структура

```
src/
  app/                 # маршруты (App Router)
    page.tsx           # Главная
    about/             # О компании (история, миссия, ценности, реквизиты)
    services/          # Услуги + динамические страницы [slug]
    catalog/           # Каталог / ассортимент + скачивание прайса
    brands/            # Бренды и партнёры (по каналам)
    cooperation/       # Сотрудничество + форма запроса КП
    delivery/          # Доставка и оплата
    faq/               # Вопрос-ответ (+ FAQ schema.org)
    contacts/          # Контакты, карта (OpenStreetMap), форма
    careers/           # Вакансии
    privacy/           # Политика конфиденциальности (Закон РБ № 99-З)
    api/lead/          # Приём заявок (точка интеграции e-mail/Telegram)
    sitemap.ts / robots.ts / manifest.ts / icon.svg
  components/          # UI, layout, формы, секции
  lib/
    site.ts            # ⬅ контакты, реквизиты, цифры — менять здесь
    nav.ts             # навигация и список услуг
    content.ts         # тексты услуг, каталог, бренды, FAQ, вакансии
```

## Где менять контент

- **Контакты и реквизиты** — `src/lib/site.ts` (телефоны, e-mail, адрес, УНП, банк, счёт, соцсети).
- **Цифры и факты** — массив `stats` в `src/lib/site.ts`.
- **Услуги / каталог / бренды / FAQ / вакансии** — `src/lib/content.ts`.
- **Меню** — `src/lib/nav.ts`.
- **Логотип** — `src/components/logo.tsx` (векторный, адаптируется под тему).
- **Прайс и презентация** — замените заглушки `public/price-list.pdf` и `public/presentation.pdf`.
- **OG-картинка / favicon** — `public/og-image.jpg`, `src/app/icon.svg`.

## Реализовано по ТЗ

- Все разделы карты сайта (п. 4), мега-меню услуг, хлебные крошки, кнопка «наверх».
- Формы: обратный звонок, запрос КП, обратная связь, отклик на вакансию — с согласием на обработку ПДн и антиспам-honeypot.
- Адаптивная вёрстка (десктоп/планшет/смартфон), мобильное «бургер»-меню.
- Тёмная/светлая тема, кликабельные телефоны/e-mail, кнопки мессенджеров, онлайн-карта.
- SEO: уникальные title/description, ЧПУ, `sitemap.xml`, `robots.txt`, микроразметка Organization / BreadcrumbList / FAQPage, OpenGraph.
- Правовое: страница политики конфиденциальности, согласие в формах, cookie-уведомление.
- Фирстиль на основе логотипа: чёрный `#1A1A1A` + красный акцент `#DE1720`, диагональная геометрия.

## Приём заявок

Заявки уходят в `POST /api/lead` (валидация zod + honeypot). Реальная доставка на e-mail заказчика
и (опционально) в Telegram подключается в `src/app/api/lead/route.ts` — там оставлена точка интеграции.

## Деплой

Оптимально — Vercel (нативно для Next.js) либо любой Node-хостинг (`pnpm build && pnpm start`).
Обязательно настроить HTTPS/SSL и переменные для отправки заявок.
